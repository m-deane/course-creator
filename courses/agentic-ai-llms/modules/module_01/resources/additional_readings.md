# Additional Readings

## Papers

- [Chain-of-Thought Prompting Elicits Reasoning in Large Language Models (Wei et al., 2022)](https://arxiv.org/abs/2201.11903) - Foundational paper on CoT prompting for complex reasoning
- [Self-Consistency Improves Chain of Thought Reasoning (Wang et al., 2022)](https://arxiv.org/abs/2203.11171) - Technique for improving reasoning accuracy through multiple sampling
- [Tree of Thoughts: Deliberate Problem Solving with Large Language Models (Yao et al., 2023)](https://arxiv.org/abs/2305.10601) - Advanced reasoning framework exploring multiple solution paths
- [Large Language Models are Zero-Shot Reasoners (Kojima et al., 2022)](https://arxiv.org/abs/2205.11916) - Shows "Let's think step by step" significantly improves reasoning
- [Prompting Is Programming: A Query Language for Large Language Models (Beurer-Kellner et al., 2023)](https://arxiv.org/abs/2212.06094) - Formal framework for understanding prompt structure

## Documentation

- [Anthropic Prompt Engineering Guide](https://docs.anthropic.com/claude/docs/prompt-engineering) - Claude-specific best practices for system prompts and few-shot learning
- [OpenAI Prompt Engineering Guide](https://platform.openai.com/docs/guides/prompt-engineering) - Official strategies and tactics for GPT models
- [Prompt Engineering Guide - Advanced Techniques](https://www.promptingguide.ai/techniques) - Comprehensive collection of prompting patterns
- [LangChain Prompt Templates](https://python.langchain.com/docs/modules/model_io/prompts/) - Reusable prompt structures and composition patterns

## Tutorials & Blogs

- [Anthropic's Claude Prompt Library](https://docs.anthropic.com/claude/page/prompts) - Production-ready prompt templates for common tasks
- [Cohere's Prompt Engineering Guide](https://docs.cohere.com/docs/prompt-engineering) - Multi-provider prompting strategies
- [Best Practices for Prompt Engineering (Brex)](https://github.com/brexhq/prompt-engineering) - Real-world enterprise prompting patterns
- [Few-Shot Learning with GPT-3 (OpenAI Cookbook)](https://github.com/openai/openai-cookbook/blob/main/examples/How_to_format_inputs_to_ChatGPT_models.ipynb) - Practical examples of in-context learning

- [Principled Instructions Are All You Need (PRINCIPLED)](https://arxiv.org/abs/2312.16171) - 26 evidence-based prompting principles
