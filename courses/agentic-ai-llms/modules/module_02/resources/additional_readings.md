# Additional Readings

## Papers

- [Toolformer: Language Models Can Teach Themselves to Use Tools (Schick et al., 2023)](https://arxiv.org/abs/2302.04761) - Meta's research on self-taught tool use in LLMs
- [Gorilla: Large Language Model Connected with Massive APIs (Patil et al., 2023)](https://arxiv.org/abs/2305.15334) - Training LLMs to effectively use thousands of APIs
- [HuggingGPT: Solving AI Tasks with ChatGPT (Shen et al., 2023)](https://arxiv.org/abs/2303.17580) - Framework for LLMs orchestrating multiple AI models as tools
- [WebGPT: Browser-assisted question-answering (Nakano et al., 2021)](https://arxiv.org/abs/2112.09332) - OpenAI's research on LLMs using web browsers as tools
- [ReAct: Synergizing Reasoning and Acting (Yao et al., 2022)](https://arxiv.org/abs/2210.03629) - Combining reasoning traces with action execution

## Documentation

- [Anthropic Tool Use Guide](https://docs.anthropic.com/claude/docs/tool-use) - Official Claude function calling documentation with best practices
- [OpenAI Function Calling Guide](https://platform.openai.com/docs/guides/function-calling) - GPT function calling patterns and JSON schema definitions
- [LangChain Tools Documentation](https://python.langchain.com/docs/modules/agents/tools/) - Pre-built tools and custom tool creation
- [JSON Schema Specification](https://json-schema.org/understanding-json-schema/) - Complete reference for defining tool parameters
- [Anthropic Tool Use Cookbook](https://github.com/anthropics/anthropic-cookbook/tree/main/tool_use) - Working examples of tool patterns
