# Additional Readings

## Papers

- [Retrieval-Augmented Generation for Knowledge-Intensive NLP Tasks (Lewis et al., 2020)](https://arxiv.org/abs/2005.11401) - Original RAG paper from Meta AI combining retrieval with generation
- [Lost in the Middle: How Language Models Use Long Contexts (Liu et al., 2023)](https://arxiv.org/abs/2307.03172) - Research showing LLMs struggle with information in middle of context
- [MemGPT: Towards LLMs as Operating Systems (Packer et al., 2023)](https://arxiv.org/abs/2310.08560) - Hierarchical memory management for LLM agents
- [Improving Language Models by Retrieving from Trillions of Tokens (Borgeaud et al., 2022)](https://arxiv.org/abs/2112.04426) - DeepMind's RETRO model with retrieval at massive scale
- [Dense Passage Retrieval for Open-Domain Question Answering (Karpukhin et al., 2020)](https://arxiv.org/abs/2004.04906) - Foundation of modern vector-based retrieval

## Documentation

- [LangChain Memory Documentation](https://python.langchain.com/docs/modules/memory/) - Comprehensive guide to conversation and semantic memory patterns
- [Anthropic Context Window Best Practices](https://docs.anthropic.com/claude/docs/context-windows) - Managing Claude's 200K context effectively
- [Pinecone Vector Database Docs](https://docs.pinecone.io/docs/overview) - Production vector database with filtering and metadata
- [Chroma Documentation](https://docs.trychroma.com/) - Open-source embedding database for RAG
- [LlamaIndex Documentation](https://docs.llamaindex.ai/) - Framework for building RAG applications with advanced indexing

## Tutorials & Blogs

- [Building RAG from Scratch (Anthropic Cookbook)](https://github.com/anthropics/anthropic-cookbook/tree/main/skills/retrieval_augmented_generation) - Complete RAG implementation with Claude
- [Advanced RAG Techniques (LlamaIndex)](https://docs.llamaindex.ai/en/stable/examples/node_postprocessor/CohereRerank.html) - Reranking, hybrid search, and query optimization
- [Vector Database Comparison Guide](https://www.pinecone.io/learn/vector-database/) - Choosing the right vector store for your use case
- [Chunking Strategies for RAG (Greg Kamradt)](https://github.com/FullStackRetrieval-com/RetrievalTutorials) - Practical experiments with chunk sizes and overlap
