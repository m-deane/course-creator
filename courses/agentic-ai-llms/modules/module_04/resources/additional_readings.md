# Additional Readings

## Papers

- [ReAct: Synergizing Reasoning and Acting in Language Models (2022)](https://arxiv.org/abs/2210.03629) - The original paper introducing the ReAct pattern, demonstrating how interleaving reasoning and action improves task completion
- [Chain-of-Thought Prompting Elicits Reasoning in Large Language Models (2022)](https://arxiv.org/abs/2201.11903) - Foundational work showing how prompting for step-by-step reasoning improves complex problem solving
- [Reflexion: Language Agents with Verbal Reinforcement Learning (2023)](https://arxiv.org/abs/2303.11366) - Introduces self-reflection mechanisms for agents to learn from mistakes
- [Tree of Thoughts: Deliberate Problem Solving with Large Language Models (2023)](https://arxiv.org/abs/2305.10601) - Extends chain-of-thought to explore multiple reasoning paths simultaneously

## Documentation

- [LangChain ReAct Agent Documentation](https://python.langchain.com/docs/modules/agents/agent_types/react) - Official implementation guide for ReAct agents in LangChain
- [LlamaIndex ReAct Agent Tutorial](https://docs.llamaindex.ai/en/stable/examples/agent/react_agent/) - Step-by-step guide for building ReAct agents with LlamaIndex
- [OpenAI Function Calling Guide](https://platform.openai.com/docs/guides/function-calling) - Technical documentation for implementing tool use with OpenAI models

## Tutorials & Blogs

- [Building Autonomous Agents with ReAct](https://www.promptingguide.ai/techniques/react) - Practical guide with code examples from PromptingGuide.ai
- [Anthropic's Guide to Agent Planning](https://docs.anthropic.com/claude/docs/agents) - Best practices for designing goal-oriented agents with Claude
- [Planning and Reasoning in LLM Agents (LangChain Blog)](https://blog.langchain.dev/planning-agents/) - Comparison of different planning strategies with implementation examples
- [Self-Reflection in AI Agents (Hugging Face Blog)](https://huggingface.co/blog/reflection-agents) - Techniques for building agents that critique and improve their own outputs
- [Goal Decomposition Strategies for Complex Tasks](https://www.anthropic.com/research/building-effective-agents) - Research-backed approaches to breaking down complex objectives

## Tools & Libraries

- [LangGraph](https://github.com/langchain-ai/langgraph) - Framework for building stateful, multi-actor applications with cycles and branching
- [AutoGPT Planning Module](https://github.com/Significant-Gravitas/AutoGPT) - Open-source implementation of autonomous planning and goal decomposition
