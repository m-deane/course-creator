# Module 4 Figures

This directory should contain visual diagrams for Module 4: Planning & Reasoning.

## Suggested Diagrams

### 1. ReAct Loop Flowchart
- Visual representation of the Thought → Action → Observation cycle
- Show decision points and loop conditions
- Include example annotations from a real task

### 2. Planning Strategy Comparison
- Side-by-side comparison of ReAct, Plan-and-Execute, Hierarchical, and Adaptive planning
- Timeline visualization showing when replanning occurs
- Complexity vs flexibility tradeoff diagram

### 3. Goal Decomposition Tree
- Hierarchical breakdown of a complex goal into subtasks
- Show parent-child relationships between tasks
- Highlight dependency chains and parallel execution opportunities

### 4. Self-Reflection Cycle
- Circular diagram: Execute → Evaluate → Identify Issues → Revise → Retry
- Include checkpoints for success criteria
- Show feedback loops and learning mechanisms

### 5. Reasoning Trace Visualization
- Step-by-step trace of an agent solving a multi-step problem
- Color-coded: thoughts (blue), actions (green), observations (yellow)
- Timeline with token counts and latency at each step

## Format Recommendations

- Use Mermaid diagrams for text-based, version-controllable graphics
- Create PNG/SVG exports for inclusion in presentations
- Keep diagrams simple with clear labels and minimal text
- Use consistent color schemes across all module diagrams
