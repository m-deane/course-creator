# Additional Readings

## Papers

- [Communicative Agents for Software Development (2023)](https://arxiv.org/abs/2307.07924) - ChatDev paper showing multi-agent collaboration for complex software engineering tasks
- [MetaGPT: Meta Programming for Multi-Agent Systems (2023)](https://arxiv.org/abs/2308.00352) - Framework for assigning different roles to agents in software development workflows
- [AutoGen: Enabling Next-Gen LLM Applications via Multi-Agent Conversation (2023)](https://arxiv.org/abs/2308.08155) - Microsoft's framework for multi-agent conversations and collaboration
- [Improving Language Model Negotiation with Self-Play and In-Context Learning (2023)](https://arxiv.org/abs/2305.10142) - Explores how agents can learn to collaborate through repeated interactions
- [Multi-Agent Collaboration: Harnessing the Power of Intelligent LLM Agents (2023)](https://arxiv.org/abs/2306.03314) - Survey of multi-agent systems and collaboration patterns

## Documentation

- [LangGraph Multi-Agent Documentation](https://langchain-ai.github.io/langgraph/tutorials/multi_agent/) - Official guide for building multi-agent systems with LangGraph
- [AutoGen Framework Documentation](https://microsoft.github.io/autogen/) - Microsoft's multi-agent conversation framework with examples
- [CrewAI Documentation](https://docs.crewai.com/) - Framework for orchestrating role-playing autonomous AI agents
- [Agency Swarm Documentation](https://github.com/VRSEN/agency-swarm) - OpenAI Assistants API based framework for agent collaboration

## Tutorials & Blogs

- [Building Multi-Agent Systems with LangGraph](https://blog.langchain.dev/langgraph-multi-agent-workflows/) - Step-by-step tutorial for creating agent teams
- [Multi-Agent Design Patterns (Andrew Ng)](https://www.deeplearning.ai/the-batch/agentic-design-patterns-part-4-multi-agent/) - Overview of multi-agent collaboration patterns
- [Orchestrating AI Agent Teams](https://www.anthropic.com/research/multi-agent-orchestration) - Best practices for coordinating multiple Claude agents
- [Agent Communication Protocols](https://huggingface.co/blog/agent-communication) - Technical deep-dive into message passing between agents
- [Building a Multi-Agent Code Review System](https://github.blog/2023-11-08-multi-agent-code-review/) - Real-world example of specialized agent collaboration
