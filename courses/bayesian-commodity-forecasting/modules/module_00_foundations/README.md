# Module 0: Foundations & Prerequisites

## Overview

This module ensures you have the mathematical and programming foundations needed for the course. It also introduces the commodity markets that will serve as our application domain throughout.

**Time Estimate:** 4-6 hours (varies based on background)

## Learning Objectives

By completing this module, you will:
1. Verify your probability and statistics foundation is sufficient
2. Set up the complete Python environment for the course
3. Understand the basic structure of commodity markets
4. Identify any knowledge gaps requiring review

## Completion Criteria

- [ ] Environment setup notebook runs without errors
- [ ] Diagnostic quiz score ≥ 70%

## If You Score Below 70% on the Diagnostic

Don't worry! This is a challenging course with high prerequisites. Consider:

1. **Probability gaps:** Work through Chapters 1-3 of "Statistical Rethinking" by McElreath
2. **Linear algebra gaps:** Khan Academy's linear algebra course (free)
3. **Python gaps:** Complete a NumPy/Pandas tutorial before proceeding
4. **Statistics gaps:** Review Khan Academy's statistics course

Allow 1-2 additional weeks for prerequisite review if needed.

## Next Steps

After completing this module:
1. If diagnostic ≥ 70%: Proceed to Module 1
2. If diagnostic < 70%: Complete recommended review, retake diagnostic
3. Review any specific weak areas identified by the diagnostic
